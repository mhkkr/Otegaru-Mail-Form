# Otegaru Mail Form
PHP Contact Mail Form

Functions Form を改修していて、無理なことが増えたので、
ほとんど作り替えたものになります。
今後はこちらを更新していきます。


## テンプレート例

edit.php
```php
<?= $template('edit', [
    'key' => 'お名前',
    'label' => 'おなまえ',
    'attribute' => [
        'maxlength' => '100',
        'placeholder' => '山田　太郎',
    ],
    'return_body' => '
        <div class="form-group  row">
            <label class="col-sm-2  col-form-label" for="{{ id }}">
                {{ required }}
                {{ label }}
            </label>
            <div class="col-sm-10">
                {{ tag }}
                {{ error_body }}
                {{ add_body }}
            </div>
        </div>
    ',
    'error_body' => '<p>{{ error_text }}</p>',
    'add_body' => '<p>hoge</p>',
]); ?>
```
